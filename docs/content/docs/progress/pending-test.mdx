# 待测试

- Agent 安全：自动连接参数改为 URL fragment，端点只允许 `localhost`、`127.0.0.1` 和 `::1`，链接不再关闭工具确认；需验证插件自动打开、手动连接、非法远程地址拦截和多画布切换。
- Canvas Agent：移除未开放的 Claude CLI HTTP Adapter 和系统 shell 启动路径；后续 Claude 侧边栏改用 Agent SDK。
- API Key 导入：API Key 改用 `#apiKey=` fragment，旧查询参数只提示不再导入；需验证 New API 跳转、地址栏清理和固定 Base URL 归一化。
- 媒体一致性：垃圾回收纳入生图/视频历史，删除历史后按全局引用清理；视频加入素材时复制独立 Blob，需验证历史、素材和画布之间互不误删。
- 画布一致性：项目异步恢复增加取消保护，持久化只写变化项目并在页面隐藏时刷盘；需验证快速切换大画布、连续编辑后立即关闭页面和多项目存储。
- WebDAV 同步：增加删除墓碑并在同步写回前重新合并最新本地状态；需验证跨设备删除、同步过程中继续编辑以及画布/素材/生成历史四个业务域。
- 导入安全：ZIP 改为解压过程限制，素材包增加大小、数量、字段、路径、媒体清单和字节数校验；需验证正常包、重复文件、压缩炸弹、损坏素材包和媒体大小不一致。
- 大画布性能：资源引用改为一次构建索引，节点和连线使用稳定回调；需验证大量节点下平移、缩放、拖拽、连线、配置节点引用和 `@` 资源选择。
- 工程质量：新增 Vitest 单元测试、TypeScript lint 和 GitHub Actions 质量工作流；镜像工作流按当前仓库 owner 推送并生成主应用 `latest` 标签。
- 配置与用户偏好：Base URL 改为固定下拉，只允许 `https://img.xmu.la` 和 `https://aiba.hk`，旧缓存和 URL 参数导入会自动归一到固定地址。
- 静态部署：nginx 和 Vercel 配置新增安全响应头，静态构建资源使用长期缓存，入口页面保持 no-cache。
- 前端加载：主要路由页面改为懒加载，需验证首页、图片、视频、素材、提示词库、画布等路由首次进入和刷新。
- 画布导入导出：zip 导入新增大小、文件数、解压体积和 `projects.json` 结构校验，需验证正常包、缺失媒体文件和超限包提示。
- 浏览器存储：画布项目改为按项目分 key 存储；大状态写入 IndexedDB 失败时不再兜底写入 localStorage，需验证存储空间不足或隐私模式提示。
- 媒体存储：图片/音视频覆盖和清理时同步释放对象 URL，需验证素材删除、画布删除和重复上传后的预览状态。
