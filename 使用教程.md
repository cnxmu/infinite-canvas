# 无限画布使用教程

本文档按实际操作顺序介绍从首次配置到完成图片、视频和画布工作流的过程。

## 1. 首次打开

使用桌面端 Chrome、Edge 或其他现代 Chromium 浏览器访问无限画布。项目不要求登录，画布、素材、生成记录和 API 配置默认保存在当前浏览器中。

首次使用前建议确认：

- 不使用无痕模式进行长期创作。
- 浏览器为站点保留足够的本地存储空间。
- 重要项目会定期导出或同步到 WebDAV。

## 2. 配置 AI 渠道

1. 点击页面右上角的配置按钮。
2. 在渠道设置中确认只读的固定站点 `https://www.aiba.hk`。
3. 选择 OpenAI 或 Gemini 调用格式。
4. 填入该接口站点签发的 API Key。
5. 点击拉取模型列表，或手动添加接口站点支持的模型名。
6. 在模型设置中分别选择图片、文本、视频和音频默认模型。
7. 设置常用图片比例、质量、数量及视频参数。

Base URL 始终固定为 `https://www.aiba.hk`，历史缓存或 URL 中的其他地址不会生效。模型是否可用由接口站点决定，前端不会伪造模型列表。

### 从 New API 自动导入

New API 的聊天链接可以使用：

```text
https://你的画布域名#apiKey={key}
```

API Key 放在 URL fragment 中，不会作为页面请求参数发送到静态站服务器。页面读取后会清理地址栏中的敏感片段。

## 3. 使用图片工作台

1. 打开“图片创作”。
2. 选择已配置的图片模型。
3. 输入提示词，按需设置尺寸、质量和生成数量。
4. 需要参考图时上传图片，再选择图生图或参考图编辑流程。
5. 提交后在生成记录中查看进度和结果。
6. 对结果执行下载、加入素材或继续编辑。

生成请求由浏览器直接发送给固定接口站点。关闭页面前应等待当前请求完成，避免丢失仍在内存中的生成状态。

## 4. 创建第一个画布

1. 打开“我的画布”。
2. 新建画布并填写名称。
3. 进入画布后，通过工具栏添加文本、图片、视频、音频或生成配置节点。
4. 拖动画布空白区域平移，使用滚轮缩放。
5. 拖动节点两侧连接点建立上下游关系。

画布会自动保存在浏览器中。返回项目列表后可以重命名、删除、批量选择、导入和导出项目。

## 5. 文本到图片工作流

推荐流程：

1. 新建文本节点，写入创作想法或原始提示词。
2. 选中文本节点，打开节点编辑面板，让文本模型扩写或优化内容。
3. 点击文本节点工具栏中的“生图”。
4. 系统会在右侧创建生成配置节点并连接文本节点。
5. 在配置节点中确认模型、比例、质量和数量。
6. 运行生成，结果会以图片节点或图片组形式插入画布。
7. 选择满意结果作为主图，下载、加入素材或继续连接到下游节点。

上游文本会作为提示词，上游图片会作为参考图。配置节点支持调整输入顺序，适合组合多个文本和媒体参考。

## 6. 图片编辑与延伸

图片节点支持：

- 下载原图。
- 保存到“我的素材”。
- 裁剪后创建新节点。
- 本地多角度变换后创建新节点。
- 作为参考图连接到生成配置节点。
- 根据节点记录的模型和提示词重试生成。

如果重试依赖的参考图已经删除，页面会提示引用缺失，不会发送空参考请求。

## 7. 视频和音频

### 视频工作台

1. 打开“视频创作”。
2. 选择视频模型，填写提示词。
3. 设置比例、清晰度、时长、声音和水印选项。
4. 按模型能力加入图片、视频或音频参考。
5. 提交并等待异步任务完成。

OpenAI 风格视频接口和 Seedance 2.0 代理流程由接口站点提供。Seedance 模型没有出现在模型列表时，可以手动填写接口站点实际支持的模型名。

### 画布中的媒体节点

- 可上传本地视频和音频到画布。
- 视频节点可直接播放、暂停和拖动进度。
- 音频节点可作为生成参考或播放素材。
- 文本、图片、视频和音频节点可以连接到生成配置节点。
- 远程视频无法因 CORS 下载时，页面可能只能保留有有效期的远程地址。

## 8. 使用画布助手

1. 在画布中选择希望助手理解的节点。
2. 展开右侧助手面板。
3. 输入问题、修改要求或生图需求。
4. 助手会把当前节点及相关上游节点作为上下文。
5. 将生成的文字或图片插回画布继续编排。

助手支持历史会话、重试、删除会话和粘贴图片。引用内容较多时，应先整理节点关系，避免把无关上游内容一并发送给 AI。

## 9. 使用本地 Canvas Agent

本地 Agent 可让 Codex 或 Claude Code 通过 MCP 读取和操作当前画布。

```bash
npx -y @basketikun/canvas-agent
```

启动后终端会显示本机地址和连接 token。进入具体画布，点击右上角 Agent，填入地址与 token 后连接。

安全边界：

- Agent 默认只监听回环地址。
- 网页只接受 `localhost`、`127.0.0.1` 或 `::1` Agent 端点。
- 首次连接后 Agent 会记录网页 Origin。
- Agent 发起的画布写操作仍需在网页侧确认。

Codex App 插件的完整安装方式见 [插件说明](plugins/infinite-canvas/README.md)，Canvas Agent 细节见 [Agent 文档](canvas-agent/README.md)。

## 10. 管理我的素材

“我的素材”支持文本、图片和视频：

1. 从提示词库、工作台或画布把内容加入素材。
2. 编辑标题、标签、来源、备注和正文。
3. 使用关键词和类型筛选快速查找。
4. 在画布中直接插入选中的素材。
5. 定期导出素材 ZIP 作为离线备份。

删除素材后，只有在画布、生成记录和其他素材中都不再引用对应媒体时，本地文件才会被清理。

## 11. 提示词库

提示词库由浏览器直接读取内置 GitHub 来源并缓存到 IndexedDB。可以按标题、标签和来源筛选，查看详情后复制提示词或加入素材。

首次加载失败时检查浏览器网络、GitHub 可访问性和站点 CSP。远程提示词源不可用不会影响已保存在本地的画布。

## 12. WebDAV 同步

1. 打开配置中的 WebDAV 页面。
2. 填写 HTTPS 地址、用户名、密码或应用密码、远程目录。
3. 测试连接。
4. 启动同步并等待所有业务域和媒体文件完成。

同步覆盖画布、素材、图片工作台记录和视频工作台记录。系统按 ID 和更新时间合并，并记录删除墓碑，减少其他设备恢复已删除数据的情况。

WebDAV 不是实时协同编辑。不要在多台设备同时高频修改同一项目；同步前后检查结果，并保留 ZIP 备份。

## 13. 导入和导出

### 画布

- 可导出单个或多个项目为 ZIP。
- ZIP 包含项目 JSON、媒体清单和关联媒体文件。
- 导入会校验压缩包大小、文件数量、路径、结构和媒体字节数。

### 素材

- 可把素材及关联媒体导出为 ZIP。
- 导入时会校验素材结构和媒体清单。

不要手动修改 ZIP 内部路径或存储键。来自未知来源的包即使通过结构校验，也应确认内容可信后再使用。

## 14. 常用快捷键

| 操作 | 快捷键 |
| --- | --- |
| 框选 | `Ctrl / Cmd` + 拖动 |
| 追加选择 | `Shift / Ctrl / Cmd` + 点击 |
| 全选 | `Ctrl / Cmd` + `A` |
| 复制、粘贴 | `Ctrl / Cmd` + `C`、`Ctrl / Cmd` + `V` |
| 撤销 | `Ctrl / Cmd` + `Z` |
| 重做 | `Ctrl / Cmd` + `Shift` + `Z` 或 `Ctrl / Cmd` + `Y` |
| 删除 | `Delete / Backspace` |
| 取消选择或关闭浮层 | `Esc` |

更完整的节点说明见 [画布节点操作手册](docs/content/docs/canvas/canvas-node-manual.mdx) 和 [画布快捷键](docs/content/docs/canvas/canvas-shortcuts.mdx)。
